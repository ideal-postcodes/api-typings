# Ideal Postcodes API Typings

Typings are exported to `index.d.ts`

### Usage

Install package

```bash
npm install --save-dev @ideal-postcodes/api-typings
```

Import interfaces and other definitions

```javascript
import { Address } from "ideal-postcodes-typings";
```

## Testing

```bash
npm test
```
